## 🌟 About the Game

**The Last LightKeeper** is a browser-based 2D game built with JavaScript and a custom ECS (Entity-Component-System) engine. Journey through mysterious realms, collect orbs, and restore the ancient lights before darkness consumes everything.

### Features

- ⚡ Custom-built ECS (Entity-Component-System) engine
- 🎨 Dynamic, atmospheric background themes per level phase
- 🔮 Orb collection & phase progression system
- 💡 Smooth character movement with polished physics
- 🎵 Ambient audio system
- 🕹️ Keyboard controls

### Controls

| Key | Action |
|-----|--------|
| `←` / `→` or `A` / `D` | Move left / right |
| `Space` or `W` / `↑` | Jump |
| `Shift` | Sprint |

## 🛠️ Tech Stack

- **Language**: JavaScript (ES Modules)
- **Bundler**: Vite
- **Architecture**: Custom ECS Engine
- **Deployment**: Netlify

## 🚀 Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
